# @levelsio.com | techno optimist wear — Clone

A static clone of [levelsio.com](https://levelsio.com), Pieter Levels' tech merch store. Built with pure HTML, CSS, and vanilla JavaScript.

## Features

- 📱 Responsive grid layout (desktop + mobile)
- 🛒 Slide-out cart drawer with add/remove/quantity controls
- 🎨 Color-coded product placeholders (hat, tee, hoodie, mug categories)
- ⚡ Zero dependencies — no frameworks, no build step
- 🖤 Clean, minimal Shopify-style aesthetic

## Products

All products from the original store are represented, including:
- **Hats/Caps** — founder mode, vibe coder, ultrathink, skill issue, and more
- **T-Shirts** — attention is all you need, 1982 database engineering, eu/acc official
- **Hoodies & Sweatshirts** — #indiehacker, eu/acc, AGI agent, you can do the ting
- **Mugs** — eu/acc official, MRR, sad? shovel sand!, fly.pieter.com
- **Misc** — AGI agent bomber, Windows 3.1 Notepad, stickers, the Indie Maker Handbook

## Run Locally

Just open `index.html` in a browser. No server needed.

```bash
open index.html
# or
python3 -m http.server 8000
```

## Deploy

This is a fully static site — deploy anywhere:
- GitHub Pages
- Vercel
- Netlify
- Cloudflare Pages
- Any static host
